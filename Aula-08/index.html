APIs de Comunicação

Exercício 1

Crie duas páginas HTML. Umas das páginas deve abrir a outra em um iframe e enviar um array com alguns valores. A página que foi aberta no iframe deverá mostrar os itens do array em um elemento select.

Exercício 2

Crie uma página HTML contendo um cabeçalho, um parágrafo com um texto explicando o que a página faz, um botão e uma área de texto. Crie um arquivo de texto com algum conteúdo. Ao clicar no botão, carregue o conteúdo do arquivo de texto na área de texto da página usando XMLHttpRequest. Para fazer esse exercício, coloque a página html, o arquivo de texto e o arquivo javascript (se for o caso) em um servidor http para que o XMLHttpRequest consiga carregar o arquivo de texto.
